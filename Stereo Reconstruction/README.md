## Installation
In this implemenation specifically, opencv version 3.4.2.16 is used
```
pip3 install opencv-python==3.4.2.16
pip3 install opencv-contrib-python==3.4.2.16
```
To run
```
cd $PATH
python3 Stereo_Reconstruction.py
```
